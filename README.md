# 🧪 LabStock PWA — Team Benny

A Progressive Web App (PWA) for tracking laboratory inventory. Works on iPhone like a native app.

## 📁 Files in this folder

```
labstock-pwa/
├── index.html       ← Main app
├── manifest.json    ← PWA config (name, icon, colors)
├── sw.js            ← Service worker (offline support)
└── icons/
    ├── icon-180.png ← iPhone home screen icon
    ├── icon-192.png ← Android icon
    └── icon-512.png ← Splash screen icon
```

## 🚀 How to deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `labstock`)
2. Upload ALL files keeping the folder structure:
   - `index.html` → root
   - `manifest.json` → root
   - `sw.js` → root
   - `icons/icon-180.png` → icons folder
   - `icons/icon-192.png` → icons folder
   - `icons/icon-512.png` → icons folder
3. Go to **Settings → Pages → Source → main branch → Save**
4. Your app will be live at: `https://YOUR-USERNAME.github.io/labstock/`

## 📱 Install on iPhone

1. Open the link above in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (box with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **Add**
5. The LabStock icon now appears on your home screen ✅

It will open full screen, work offline, and feel like a native app!
